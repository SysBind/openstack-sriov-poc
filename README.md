# OpenStack SR-IOV POC

## Requirements
- Linux Machine with _enough_ CPU & Memory
- Nested Virtualization Enabled
- Libvirt & Qemu
- Vagrant


# Running
- ```vagrant plugin install vagrant-libvirt```
- ```vagrant up```
